================================================
:pep:`265`: Sorting Dictionaries
================================================
.. currentmodule:: pep265

:func:`.itemlist` sorts a dictionary and returns its item list::

    >>> from rejected_peps import pep265 as p
    >>> d = {1: 4, 2: 3, 0: 6}
    >>> p.itemlist(d, sortby=p.ORIGINAL)  # Untouched
    [(1, 4), (2, 3), (0, 6)]
    >>> p.itemlist(d, sortby=p.KEYS)  # Keys
    [(0, 6), (1, 4), (2, 3)]
    >>> p.itemlist(d, sortby=p.VALUES)  # Values
    [(2, 3), (1, 4), (0, 6)]

================================================
:pep:`276`: Simple Iterator for ints
================================================
.. currentmodule:: pep276

:class:`.int` makes an integer iterable, beijng equivalent to ``range(x)``::
    >>> from rejected_peps import pep276 as p
    >>> range(p.int(10))
    range(0, 10)
    >>> range(p.int(-10))
    range(0, -10)

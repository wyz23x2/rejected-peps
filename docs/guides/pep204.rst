============================
:pep:`204`: Range Literals
============================
.. currentmodule:: pep204

You can create :class:`range` objects quickly with pep204::

    >>> from rejected_peps import pep204 as r
    >>> r[1:5]
    range(1, 5)
    >>> r[4:17:4]
    range(4, 17, 4)
    >>> r[6]
    range(6)

.. seealso:: 

    Module :mod:`pep212`
        :func:`pep212.indices` that is the same as ``range(len(x))``.
